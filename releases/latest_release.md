# Introducing Smart Lookup Core `v0.1`

> [!NOTE] What's new in `v0.1.4`
> fix: use window.clearTimeout for consistency in create_debounced_submit

Smart Lookup is now its own Core plugin for Obsidian.

Use Lookup when you remember the idea, but not the exact phrase, filename, or heading. Ask naturally, scan the top semantic results, expand the strongest matches, then open, link, or send the useful notes into your next workflow.

## Why this matters

Obsidian search is still the right tool when you know the exact words.

Smart Lookup is for a different moment: the note exists somewhere in your vault, but your memory does not match the words you used when you wrote it.

Use the simple boundary:

- Current note -> Smart Connections
- Question -> Smart Lookup
- Exact phrase -> Obsidian search

That keeps each surface clear. Connections helps while you are working from the current note. Lookup helps when your question or idea is the anchor.

## What's included

Smart Lookup has been extracted from Smart Connections into its own plugin path.

That makes the workflow easier to explain and easier to choose:

- Smart Connections surfaces related notes while you write.
- Smart Lookup searches by meaning from a question or idea.
- Obsidian search handles exact phrases, filenames, headings, tags, and operators.
